- Spring Data JPA
- JWT

- RabbitMQ
### Container
- Docker
- Kubernetes
### API Gateway
- Spring Cloud Gateway
### Database
- H2 Database
- MySQL(MariaDB)

- 모니터링

- 구성

	- (Istio)
- 분산 추적

- Configuration
	- Spring Cloud Config
	- Github(Native File)
- 서비스간 통신
	- FeignClient (RestTemplate)
